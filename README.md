## Installation

```bash
$ npm install
```

## Setup Database


set up an env  and replace DB_URL with the actual  url for your database

`postgres://xxxxxxxxxxxxxx`


## Documentation

start the application in dev 
```bash
$ npm run start
```

 head to  view the swagger documentation


http://localhost:3000/api




## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
